# STAT 302: Final Project — PSPS

## Overview  
This project explores demographic and socioeconomic patterns in the Philippines using data from the **Philippines Socioeconomic Panel Survey (PSPS)**. The data was directly provided by the Global Poverty Research Lab (GPRL).  

The analysis focuses on **migration trends across age, gender, and socioeconomic characteristics**, translating complex survey data into clear, interpretable visual insights.

---

##  Folders

- `App-final`: final rendered outputs (HTML dashboard/app)
- `Report`: written analysis and executive summary files (Quarto `.qmd`)
- `scripts`: R scripts for data cleaning, analysis, and visualization
